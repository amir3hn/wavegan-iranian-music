# WaveGAN for Iranian Classical Music

This repository contains PyTorch-based WaveGAN experiments developed as part of my Master's thesis:

**Music Generation in Iranian Classical Music Using Neural Networks**

## Overview

The project explores audio-based music generation using WaveGAN and Wasserstein GAN with gradient penalty.

## Technologies

- Python
- PyTorch
- Librosa
- NumPy
- Matplotlib

## Main Components

- `models.py`: WaveGAN generator and discriminator architectures
- `train.py`: training loop using WGAN-GP
- `utils.py`: audio loading, preprocessing, visualization, and sample saving
- `params.py`: training configuration

## Research Context

This work investigates how deep generative models can be used for culturally grounded music generation, with a focus on Iranian classical music.
